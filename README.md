# OcTcProfileScraper

Returns user information from their profile at [http://www.oc.tc/](http://www.oc.tc).

## Usage

From the command line:
`python ocpscraper.py <username>`

With python:
```python
import ocpscraper
print ocpscraper.get_user_info('bcbwilla')
```
